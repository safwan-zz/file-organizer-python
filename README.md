# File Organizer (Python)

This Python script automatically organizes files in a folder based on their extensions.

## Features
- Creates folders for each file type
- Moves files into the corresponding folder
- Handles files without extensions

## How to Use
1. Run `main.py` or `main.exe`
2. Enter the folder path
3. Files are automatically sorted

## Technologies
- Python 3.14
- os & shutil modules
